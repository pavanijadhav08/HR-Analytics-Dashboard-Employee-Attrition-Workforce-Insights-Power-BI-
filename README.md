# HR Analytics Dashboard – Employee Attrition & Workforce Insights (Power BI)


## Project Objective

- The objective of this project is to develop an interactive HR Analytics Dashboard using Power BI to analyze employee data, with a primary focus on attrition, workforce demographics, salary distribution, and job roles.

- This dashboard helps organizations understand employee turnover patterns and supports data-driven HR decision-making to improve retention and workforce planning.

## Business Questions & KPIs
- What is the overall employee attrition rate?
- Which age groups have the highest attrition?
- How does attrition vary across departments and job roles?
- What is the impact of salary levels on attrition?
- Which education fields contribute most to employee turnover?
- How does experience (years at company) affect attrition?

## Process
1. Data Collection
 - HR dataset including:
  - Employee demographics
  - Salary details
  - Job roles & departments
  - Education background
  - Years of experience
  - Attrition status
    
2. Data Cleaning & Transformation
 - Handled missing and inconsistent values
 - Created calculated columns for:
   - Age groups
   - Salary slabs
   - Experience categories
 - Used DAX to build KPIs and measures
   
 3. Data Modeling
 - Built relationships between HR attributes
 - Optimized data structure for faster reporting
 - Enabled filtering by department and job roles
   
 4. Dashboard Development
 - Designed an interactive dashboard with:
  - KPI cards (Employees, Salary, Age, Experience)
 - Attrition analysis by:
  - Age
  - Salary
  - Job role
  - Education
 - Department-level filters (HR, R&D, Sales)
   
 5.  Visualization Design
 - Donut chart → Attrition by Education
 - Bar charts → Attrition by Age, Salary, Job Role
 - Line chart → Attrition trend by years
 - KPI cards → Key workforce metrics
 - Applied a modern gradient UI for better visual appeal
